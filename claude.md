# CLAUDE.md

## Project Purpose

A lab project exploring AI coding assistant behavior and rule enforcement.

## Tech Stack

- Frontend: React + TypeScript (strict mode)
- Styling: Tailwind CSS
- Testing: Vitest
- Backend: FastAPI (Python)

## Naming Conventions

1. Components: PascalCase (e.g., `UserCard.tsx`)
2. Functions/variables: camelCase (e.g., `handleClick`)
3. CSS classes: Tailwind utility classes only, no custom class names

## Pattern to Always Avoid

- Never use inline styles (`style={{}}`) — use Tailwind classes instead
